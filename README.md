## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## General context
Have quickly integrated checkout business logic by copying and pasting code from the other repo to `checkout` directory (this was literally a 10min job with complete disregard for quality, testing, etc.).

Checkout processes are stored in memory. Stopping the application will clear the persistence.

### How to use
* Creating a checkout - `POST http://localhost:3000/checkout`
* Adding a discounted item to the checkout (products `id: 1` are set up to be discounted) - `PUT http://localhost:3000/checkout/$checkoutId`
```json
{
  "id": "1",
  "name": "Classic Ad",
  "description": "Offers the most basic level of advertisement",
  "retailPrice": 26999
}
```
* Adding another item to the checkout - `PUT http://localhost:3000/checkout/$checkoutId`
```json
{
  "id": "some-other-non-discounted-product",
  "name": "Classic Ad",
  "description": "Offers the most basic level of advertisement",
  "retailPrice": 26999
}
```
* Getting a total - `GET http://localhost:3000/checkout/total/$checkoutId`
